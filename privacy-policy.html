Chrome Dashboard — 개인정보처리방침

최종 수정일: 2026-06-14

Chrome Dashboard(이하 "본 확장")는 크롬 새 탭을 개인용 대시보드로 대체하는 브라우저 확장 프로그램입니다. 본 확장은 사용자의 데이터를 개발자의 서버로 수집·전송·판매하지 않습니다. 모든 데이터는 사용자의 브라우저와 사용자 본인의 Google 계정 안에만 머무릅니다.

1. 처리하는 데이터와 목적

데이터출처 / 권한목적저장 위치
위젯 레이아웃·설정·테마·배경화면storage대시보드 구성 유지브라우저 로컬 (chrome.storage)
북마크bookmarks바로가기 위젯 표시처리만 함, 저장·전송 없음
Google 프로필·이메일OAuth userinfo.profile/email로그인 계정 표시브라우저 로컬 캐시
Google 캘린더 일정OAuth calendar.events캘린더 위젯에서 일정 보기·추가사용자 Google 계정 (저장 안 함)
Google Tasks 할 일OAuth tasks할 일 위젯에서 보기·추가사용자 Google 계정 (저장 안 함)
메모 파일OAuth drive.file메모를 사용자 Drive의 보이는 파일로 동기화사용자 Google Drive
위치(위경도)사용자 입력 / 브라우저날씨·일출일몰 조회로컬, Open-Meteo 호출 시에만 전송

2. 제3자 전송

본 확장은 다음 외부 API를 사용자의 브라우저에서 직접 호출합니다. 호출은 해당 기능을 켠 경우에만 발생하며, 개발자 서버를 경유하지 않습니다.

  • Google APIs (googleapis.com) — 캘린더·Tasks·Drive·프로필. Google 개인정보처리방침
  • Open-Meteo — 날씨/일출일몰 (위경도만 전송, 계정 정보 없음)
  • Yahoo Finance — 시세 위젯의 공개 시세 데이터

본 확장은 사용자의 Google 데이터를 광고·머신러닝 학습 등 명시된 기능 외 목적으로 사용하지 않으며, 제3자에게 양도·판매하지 않습니다. 이는 Google API Services User Data Policy(Limited Use 요건 포함)를 준수합니다.

3. 데이터 보관 및 삭제

설정·레이아웃은 확장을 삭제하면 함께 제거됩니다. Google 계정 연동은 확장 내 로그아웃 또는 Google 계정 권한 페이지에서 언제든 해지할 수 있습니다. Drive에 저장된 메모 파일은 사용자가 Drive에서 직접 삭제할 수 있습니다.

4. 아동

본 확장은 만 13세 미만 아동을 대상으로 하지 않습니다.

Privacy Policy (English Summary)

Chrome Dashboard replaces the new-tab page with a personal dashboard. It does not collect, transmit, or sell your data to the developer. All data stays in your browser and your own Google account. Google data (Calendar, Tasks, Drive via drive.file, profile/email) is accessed only to power the widgets you enable and is never used for advertising or model training, in compliance with the Google API Services User Data Policy, including the Limited Use requirements. Weather/sunrise use Open-Meteo (coordinates only); the markets widget uses public Yahoo Finance quotes. You can revoke access anytime via the in-app logout or your Google account permissions.
